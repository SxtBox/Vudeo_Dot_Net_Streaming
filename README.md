# Vudeo Dot Net Streaming APIS

# API Description
    You can build playlist from url or from json file,
    you can also improvise this API on Your panel to play direct videos as mp4 or Web player.
    
# API Supports Multi Formats Data
    Smart TV
    VLC
    Kodi
    Xtream Codes Panel
    And Web Player
